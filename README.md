# QuadraFácil

Sistema em Java para gerenciamento de reservas de quadras esportivas.

## Problema

A ausência de um sistema organizado para reservas de quadras esportivas pode causar conflitos de horários, perda de informações e dificuldade no controle dos espaços disponíveis.

## Objetivo

Desenvolver uma aplicação Java com interface em console que permita cadastrar usuários, quadras e modalidades, além de realizar e gerenciar reservas.
